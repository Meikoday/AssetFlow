# 电脑资产管理系统 - 客户端

这是电脑资产管理系统的前端界面，基于 Vue 3 和 Element Plus 开发。

## 功能特点

- 美观的用户界面
- 响应式布局
- 资产信息的可视化展示
- 详尽的硬件信息展示

## 安装和构建

```bash
# 安装依赖
npm install

# 构建生产版本
npm run build
```

## 主要页面

- **仪表盘** - 系统概览和资产统计
- **资产列表** - 所有资产的列表视图，支持搜索和排序
- **资产详情** - 单个资产的详细硬件信息

## 与服务器通信

前端通过 RESTful API 与后端服务器通信，API 地址配置在 `vite.config.js` 文件中。
开发模式下支持 API 代理，默认转发到 `http://localhost:80`。
